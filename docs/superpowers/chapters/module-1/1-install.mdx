# Install in 5 minutes

Let's get pi running on your machine. You'll need Node.js 18+ installed.

## Step 1: Install via npm

Open your terminal and run:

```bash
npm install -g @mariozechner/pi-coding-agent
```

<Callout variant="tip" title="Using Bun?">
If you prefer Bun, just run `bun add -g @mariozechner/pi-coding-agent` instead.
</Callout>

## Step 2: Configure API Key

pi supports multiple providers. The simplest is Anthropic (Claude).

```bash
# Set your Anthropic API key
export ANTHROPIC_API_KEY=sk-ant-...
```

<Callout variant="warning" title="API Key Safety">
Never commit API keys to git. Add `ANTHROPIC_API_KEY` to your shell profile (~/.bashrc, ~/.zshrc) or use a `.env` file with your projects.
</Callout>

## Step 3: Launch pi

```bash
pi
```

You should see the startup header showing loaded extensions, skills, and your current configuration.

## Alternative: Use /login

If you have a subscription (Claude Pro/Max, ChatGPT Plus, etc.), you can use the built-in OAuth:

```bash
pi
/login   # Opens provider selection
```

Select your provider and authenticate in your browser.

## Verify Installation

Let's confirm everything works:

```bash
pi --version
```

This should output the current pi version.

## Quick Test

Run this non-interactive command to verify pi can communicate with the model:

```bash
pi -p "Say hello in one sentence"
```

If you see a response, you're all set!
